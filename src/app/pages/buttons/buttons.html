BP Buttons

Componente reutilizable para acciones del sistema, con variantes visuales y estados.

1. Botones básicos

Uso estándar para acciones principales.

2. Botones soft

Variante visual más suave. Mantiene el mismo color, pero con menor impacto visual. Ideal para acciones secundarias.

3. Botones outline

Botones con borde. Se usan para acciones neutras o menos prioritarias.

4. Botones gradient

Variante destacada para llamadas a la acción (CTA).

5. Botones con icono

Se utilizan para reforzar visualmente la acción.

6. Botones solo icono

Ideales para acciones rápidas en tablas o toolbars.

7. Tamaños

Variantes de tamaño para distintos contextos.

8. Estados

Estados especiales del botón.

Guía rápida
  • Básico: acción principal
  • Soft: acción secundaria
  • Outline: acción neutra
  • Gradient: llamada a la acción
  • Icon: acciones rápidas
