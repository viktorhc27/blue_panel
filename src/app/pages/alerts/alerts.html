BP Alerts

Componentes reutilizables para mostrar notificaciones, estados y mensajes del sistema.

1. Alertas básicas

Uso recomendado para mostrar mensajes simples de estado.

2. Alertas con título

Ideales para mensajes importantes o secciones informativas.

3. Alertas cerrables

El usuario puede cerrar manualmente la alerta.

4. Alertas pequeñas (status)

Pensadas para dashboards, encabezados o estados rápidos.

5. Alertas con acción

Incluyen un botón para acciones rápidas.

6. Alertas con borde lateral

Se utilizan para destacar información importante o persistente.

7. Cards de estado

Componentes visuales para mostrar estados del sistema en dashboards.

8. Alertas inline (minimal)

Para estados compactos dentro de tablas o listas.

9. Alertas con acción (inline)

Combinan estado + acción directa.

Guía rápida
  • Alertas básicas: mensajes simples
  • Con título: mensajes importantes
  • Cerrables: feedback temporal
  • Small: dashboards
  • Card: estados del sistema
  • Acción: interacción rápida
